# Construct 3

[Construct 3](https://www.construct.net/a/260259?t=construct3) 是一款小而美的游戏开发工具，轻巧且功能强大，完全基于浏览器环境运行。

**我认为 Construct 3 是当下世界，最友好、最容易上手的 2D 游戏开发引擎。**

我们希望将它分享给更多的人，人人都能享受到游戏开发的乐趣。
